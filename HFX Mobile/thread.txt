[size=large][b]About[/b][/size]
Many users primarily browse HF from their mobile devices and the mobile experience isn't as smooth as accessing the site via desktop.
I created [b]HFX Mobile[/b] in an attempt to improve your HF mobile browsing experience, along with porting features from the [url=https://hackforums.net/showthread.php?tid=6312252]desktop browser extension, HFX[/url].
Check out how to install this script to your phone along with current features below.

Please post if you are interested, plan to install, or enjoy the extension!

[size=large][b]How to Install[/b][/size][list=1]
[*] Download and install the app "[url=https://apps.apple.com/us/app/stay-for-safari/id1591620171][b]Stay for Safari[/b][/url]"
[*] Enable the extension in Safari by navigating to "[b]Settings > Apps > Safari > General > Extensions > Stay[/b]". Enable "Allow Extension".
[*] Install this extension by opening the following link in Safari and tap "Install":
[code]
https://github.com/xadamxk/HF-Userscripts/raw/refs/heads/master/HFX%20Mobile/HFX%20Mobile.user.js
[/code]
[url=https://github.com/xadamxk/HF-Userscripts/raw/refs/heads/master/HFX%20Mobile/HFX%20Mobile.user.js]Click here if you are on your phone[/url]
[*] Activate the "HFX Mobile" script so it will be enabled the next time you visit HF in Safari
[*] That's it - open up HF and enjoy!
[/list]
